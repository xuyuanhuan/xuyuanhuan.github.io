#js的引入方式

###在一对script标签中引入js代码

##js引入放到头部或底部的区别

###头部引入
代码执行时下面标签内容没加载完的时候如果需要获取元素是会出错的，所有需要加上window.onload等页面所有内容加载完载获取就可以了。

###底部引入
代码是重上往下执行的，执行到底部的是后，body里面的所有内容都加载完了，所有在底部用JS获取元素也没有问题了。

#变量的命名规则
1.区分大小写；

2.第一个字符必须是一个字母、下划线（_）或一个美元符号（$）;

3.其他字符可以是字母、下划线、美元符号或数字；

4.不能含有空格；

5.不能以关键字或保留字命名。

关键字可用于表示控制语句的开始或结束，或者用于执行特定操作等。
按照规则，关键字也是语言保留的，不能用作标识符

#onclick事件的绑定
我们在浏览网页时,经常会通过点击鼠标来进行一些操作,比如点击登录按钮进行账 号的登录,点击注册按钮进入到一个账号注册页面,我们如何来实现一个点击时间 呢?这里我们就需要用到onclick事件绑定。
示例:点击红色div,弹出一个对话框

#onmouseover、onmouseout事件

鼠标基本事件除了onclick之外,还有:onmouseover、onmouseout事件。

onmouseover表示鼠标移入,在鼠标移入元素时会触发该方法;

onmouseout表示鼠标移出,在鼠标移出元素时,会触发该方法。

```javascript
//onmouseover/onmouseout是有BUT的

// IE8后改为用onmouseenter/onmouseleave
// onmouseenter:当鼠标移到DIV里面的时候触发的事件
div.onmouseenter = function () {        
    div.style.backgroundColor = "blue";
    alert("进来")
}
// onmouseleave:当鼠标移出div的时候触发的事件
div.onmouseleave = function () {        
    div.style.backgroundColor = "red";
    alert("出去")
}

``
